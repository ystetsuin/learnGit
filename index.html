Git diary

Cloning project from remote repo to local

  1. git init
  2. git clone repo.git
  3. Do some changes

Adding new code to remote repo

  1. git add .
  2. git commit -m "message"
  3. git commit push

Branches and Checkout

  1. git branch - show all branches
  2. git branch fix1 - create new branch "fix1"
  3. git checkout fix1 - switch to branch fix1
  4. git checkout -b fix2 - create and switch to branch fix2

Merge our branches

  1. git checkout master - switch to master
  2. git merge fix1 - add code from fix1 to master
  3. git branch -d fix1 - delete branch fix1

Get up to date code

  1. git fetch

Просмотр изменений

  1. git diff - покажет, что именно изменилось в файле: красным — удаленное, зеленым — добавленное
  2. git log - покажет всю историю коммитов этого репозитория, если она есть
