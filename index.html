Git diary

Cloning project from remote repo to local

  1. git init
  2. git clone repo.git
  3. Do some changes

Adding new code to remote repo

  1. git add .
  2. git commit -m "message"
  3. git commit push

Branches and Checkout

  1. git branch - показывает все ветки
  2. git branch fix1 - создает ветку "fix1"
  3. git branch -d fix1 - удаляет ветку "fix1"
  4. git checkout fix1 - переключает на ветку fix1
  5. git checkout master - переключает на ветку master
  6. git checkout -b fix2 - создает и переключает на ветку fix2
  7. git checkout HASH - откатить файл до определенного коммита.
  8. git checkout index.js - откатить файл до последнего коммита

Merge

  1. git merge fix1 - add code from fix1 to master

Get up to date code

  1. git fetch
  2. git pull

Просмотр изменений

  1. git diff - покажет, что именно изменилось в файле: красным — удаленное, зеленым — добавленное
  2. git log - покажет всю историю коммитов этого репозитория, если она есть
  3. git show HASH - покажет какие изменения были внесены в определенном коммите

Reset, revert

  1. git restore --staged - убрать файл из индекса (после git add, но перед git commit)
