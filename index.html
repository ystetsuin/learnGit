Git diary

Cloning project from remote repo to local

  1. git init

  2. git clone repo.git

  3. Do some changes

Adding new code to remote repo

  1. git add .

  2. git commit -m "message"

  3. git commit push

Branches and Checkout

  1. git branch - show all branches

  2. git branch dev1 - create new branch "dev1"

  3. git checkout dev1 - switch to branch dev1

  4. git checkout -b dev2 - create and switch to branch dev1
